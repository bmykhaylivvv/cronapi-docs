---
title: "Create cron"
api: "POST /crons"
description: "Create a new recurring scheduled webhook"
keywords: ["create cron", "new cron", "schedule webhook"]
---

## Request body

<ResponseField name="cron" type="string" required>
  Cron expression defining the schedule (e.g., `*/5 * * * *`). Must be a valid 5-field cron expression.
</ResponseField>

<ResponseField name="url" type="string" required>
  Webhook URL to call on each execution.
</ResponseField>

<ResponseField name="name" type="string">
  Optional display name for the cron.
</ResponseField>

<ResponseField name="method" type="string" default="POST">
  HTTP method: `GET`, `POST`, or `DELETE`.
</ResponseField>

<ResponseField name="headers" type="object" default="{}">
  Custom HTTP headers to include in the webhook request.
</ResponseField>

<ResponseField name="body" type="object" default="{}">
  JSON body to send with the webhook request (used with POST).
</ResponseField>

## Response

Returns the created cron object with a `201` status code.

<CodeGroup>
```bash cURL
curl -X POST https://cronapi.dev/api/v1/crons \
  -H "Authorization: Bearer your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Daily report",
    "cron": "0 9 * * *",
    "url": "https://example.com/report",
    "method": "POST",
    "headers": {"X-Custom": "value"},
    "body": {"notify": true}
  }'
```

```javascript Node.js
const response = await fetch("https://cronapi.dev/api/v1/crons", {
  method: "POST",
  headers: {
    "Authorization": `Bearer your_api_key_here`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: "Daily report",
    cron: "0 9 * * *",
    url: "https://example.com/report",
    method: "POST",
    headers: { "X-Custom": "value" },
    body: { type: "daily" },
  }),
});
const cron = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://cronapi.dev/api/v1/crons",
    headers={"Authorization": "Bearer your_api_key_here"},
    json={
        "name": "Daily report",
        "cron": "0 9 * * *",
        "url": "https://example.com/report",
        "method": "POST",
        "headers": {"X-Custom": "value"},
        "body": {"notify": true},
    },
)
cron = response.json()
```
