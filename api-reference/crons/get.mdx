---
title: "Get cron"
api: "GET /crons/{id}"
description: "Retrieve a single cron by ID"
keywords: ["get cron", "fetch cron", "cron details"]
---

## Path parameters

<ResponseField name="id" type="string" required>
  The UUID of the cron to retrieve.
</ResponseField>

## Response

Returns the cron object if it exists and belongs to the authenticated user.

<CodeGroup>
```bash cURL
curl https://app.cronapi.dev/api/v1/crons/a1b2c3d4-e5f6-7890-abcd-ef1234567890 \
  -H "Authorization: Bearer your_api_key_here"
```

```javascript Node.js
const response = await fetch(
  `https://app.cronapi.dev/api/v1/crons/${cronId}`,
  { headers: { "Authorization": `Bearer your_api_key_here` } }
);
const cron = await response.json();
```

```python Python
import requests

response = requests.get(
    f"https://app.cronapi.dev/api/v1/crons/{cron_id}",
    headers={"Authorization": "Bearer your_api_key_here"},
)
cron = response.json()
```
